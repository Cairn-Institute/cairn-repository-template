# cairn-repository-template

## Project purpose

This is the standard starter template for new CAIRN Institute repositories. Create new repositories from this template rather than starting from an empty repository, so that every project begins with consistent structure, documentation, and governance files.

## Mission alignment

Consistent repository structure supports CAIRN Institute's ability to build technology professionally, securely, transparently, and sustainably across educational, workforce, technology, research, and community-impact projects.

## Project status

Template. Not deployed.

## Intended users

CAIRN Institute developers, technical leads, students, interns, and approved contractors starting new repositories.

## Major features

This template provides a starter README, an AGENTS.md file for Codex and other AI coding assistants, contribution and security guidance, a CODEOWNERS file, a changelog, ignore rules, an environment variable example file, and placeholder docs, src, and tests directories.
